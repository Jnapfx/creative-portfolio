# The Exchange Preview Hub

This folder contains two early webpage mockup directions for **The Exchange**, a pilot initiative connected to CT Tech Hub.

The purpose of this preview hub is to allow the team and committee to compare two possible directions before moving into final design and development.

## Mockup Options

### Draft A — Informational / Form-Based Direction

This version presents The Exchange as a more informational pilot landing page. It focuses on explaining the initiative, outlining how the process works, and guiding employers, talent, and partners to submit interest through forms.

This direction does not publicly display specific opportunities.

### Draft B — Opportunities Visible Direction

This version presents The Exchange with selected pilot opportunities shown as preview cards. It gives users a clearer idea of the types of opportunities available while still keeping the process guided and structured.

This direction is more visual and interactive, but it is not intended to function as a full job board or open marketplace.

## Project Status

These mockups are early concept drafts created for review and discussion. They are not final designs or production-ready builds.

## Preview Structure

* `/` — Preview hub landing page
* `/draft-a/` — Informational / form-based mockup
