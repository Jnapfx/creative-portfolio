The Exchange Pilot Webpage Mockups

Below are two early concept directions for review. Both are rough drafts intended to help the committee compare the overall webpage approach before final design and functionality are refined.

Review note: Draft A keeps the pilot more informational and form-driven. Draft B makes selected pilot opportunities more visible while still keeping the process guided and structured.
Option A

Informational / Form-Based Direction

This version presents The Exchange as an official pilot landing page. It explains the program, keeps opportunities high-level, and guides users to submit interest through forms.

  • More controlled and committee-safe
  • No public opportunity listings
  • Best for a guided pilot intake process
View Draft A
Option B

Opportunities Visible Direction

This version shows selected pilot opportunity preview cards so users can better understand what they may engage with, while still keeping follow-up managed by the program team.

  • More visual and user-facing
  • Shows sample pilot opportunity cards
  • Best for helping users quickly understand value
View Draft B

Quick Comparison

Draft A Explains the pilot and uses forms as the main interaction. Opportunities are discussed as general categories.
Draft B Shows selected opportunity previews/cards so visitors can browse examples before expressing interest.
Decision Point Whether the committee prefers a more controlled informational page or a more visible opportunity showcase.
