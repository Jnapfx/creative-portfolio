# Minimal Portfolio

A minimalist, Apple-style portfolio website designed for a Digital Designer & Editor.

## Features

- **Minimalist Design**: Clean typography, whitespace, and subtle animations.
- **Responsive**: Fully responsive layout for desktop, tablet, and mobile.
- **Interactive Gallery**:
  - **Project Modal**: Detailed view of projects.
  - **Image Navigation**: "Previous" and "Next" buttons to cycle through images.
  - **Keyboard Support**: Use Arrow keys to navigate gallery.
  - **Touch Gestures**: Swipe left/right on mobile to change images.
  - **Smooth Transitions**: 1:1 touch drag and smooth slide animations.

## Technologies

- HTML5
- CSS3 (Variables, Flexbox, Grid)
- Vanilla JavaScript

## How to Run Locally

1.  Clone the repository.
2.  Open `index.html` in your browser.

## Deployment

This project is ready for GitHub Pages.

1.  Push the code to a GitHub repository.
2.  Go to **Settings** > **Pages**.
3.  Select the `main` branch as the source.
4.  Save.
